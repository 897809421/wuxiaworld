# 爬取 wuxiaworld 小说

默认爬取斗破苍穹英文全章节

使用说明：
- 首先你得有 node
- 到 wuxia 根目录
- npm i 安装依赖
- node wuxiaworld.js 执行
- 执行完毕后目录里就有小说了
